## Komentorivitoiminnot
Testit suoritetaan komennolla: `mvn test`

Testikattavuus:
``
mvn test jacoco:report
``

Checkstyle:`mvn jxr:jxr checkstyle:checkstyle`

jar-tiedoston genenerointi: `mvn package`

javadoc: `mvn javadoc:javadoc`
